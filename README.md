# CountTogether plugin for Macro Deck 2
Control your [CountTogether](https://counttogether.app) counters with Macro Deck!

# Features
- ### Increment/Decrement Counter
Increment and decrement counters with a press of a button in Macro Deck.
- ### Variables
Show the current value of counters on your buttons.

### This is a plugin for [Macro Deck 2](https://github.com/Macro-Deck-app/Macro-Deck), it does NOT function as a standalone app
